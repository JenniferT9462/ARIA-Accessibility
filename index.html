ARIA-Accessibility

Using aria-label on a input tag

The aria-label should be used to provide a text alternative to an element that has no visible text on the screen.

If you use a label tag on a input

The input in the label tags would not require a aria-label

A button with an aria-label attribute

Used to replace an existing label with more precise information

A link with a role attribute

The button role is for clickable elements that trigger a response when activated by the user. It tells the screen reader the element is a button.
